### 3.2 Детальный разбор по каждому фреймворку

#### LangChain / LangGraph
- **Архитектура**: DAG-граф (LangGraph), chains, Runnable, LCEL
- **Multi-Agent**: Supervisor + worker via LangGraph (новое в 2025)
- **Ключевая фича**: LangSmith (observability), Hub, Templates
- **Слабые места**: Сложность API, breaking changes, абстракции «утекают»

#### CrewAI
- **Архитектура**: Ролевая модель (Agent → Task → Crew → Process)
- **Multi-Agent**: Нативный; agents с ролями, goals, backstory; sequential/hierarchical process
- **Ключевая фича**: Простота, role-based дизайн, встроенный delegation
- **Слабые места**: Меньше интеграций, молодой, ограниченная кастомизация

#### AutoGen (Microsoft)
- **Архитектура**: AgentChat, групповые чаты, publish-subscribe, ручное управление flow
- **Multi-Agent**: Conversational; agents общаются через сообщения; GroupChatManager
- **Ключевая фича**: Гибкость, code execution, человеческий-in-the-loop
- **Слабые места**: Низкоуровневый, много boilerplate, документация отстаёт
